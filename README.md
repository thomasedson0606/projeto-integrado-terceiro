# projeto-integrado-terceiro
#Descrição
O projeto tem como objetivo propor um sistema de agendamento para um salão de beleza fictício. A ideia é facilitar a organização dos horários, dos clientes e dos serviços oferecidos, ajudando o salão a evitar conflitos de agenda e melhorar o atendimento.

#Funcionalidades
Cadastro de clientes
Login no sistema
Agendamento de serviços 
Vizualização da agenda do salão

#Tecnologias utilizadas
HTML
CSS
JavaScript
PHP
